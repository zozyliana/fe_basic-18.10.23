Команди Git у терміналі

Збірка зображень
  • git add - приймає шлях файлу чи директорії.
  • git branch - додавання нової гілки до існуючої або видалення з неї.
  • git checkout - перехід до існуючої гілки або створення нової.
  • git clone - копія вихідного коду з віддаленого сховища.
  • git commit - зберігає зміни у вашому місцевому сховищі.
  • git init - створення нового сховища.
  • git merge - об'єднання вашої гілки з батьківським
  • git pull - отримує останні завантажені зміни з віддаленого сервера в локальний.
  • git push - надіслання зафіксованих зміни файлу з локального репозиторію на віддалений.
  • git status - вказує на стан робочого каталогу та місця проведення.
